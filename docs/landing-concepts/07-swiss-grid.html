§ 01 / The system

Verification / Evidence / Discipline.

01
Verification

Behavior is the signal. Activity is the noise.

Signer disengagement. Champion airtime collapse. Procurement entered late. Competitor introduced during pricing. Mallin measures what the deal is actually doing — not what the CRM says it is.

02
Evidence

Every claim has a source. Every source has a timestamp.

Quoted sentence, exact timestamp, named speaker. Paraphrased evidence is a trust failure. Every surface — pre-call brief, Slack alert, manager DM, CRM update — renders the same five-element sequence: risk, move, evidence, temporal proof, accountable decision.

03
Discipline

Trust is earned in stages. Never auto-written: four fields.

Suggest → Apply → Auto+Confirm. The agent never writes Stage, Amount, Close Date, or Forecast Category. Not after trust is earned. Not when statistically safe. The constraint is structural, not configurable.
