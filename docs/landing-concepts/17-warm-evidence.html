02How it stays responsible

Earned reliance scales. Assumed reliance fails under load.

Stage Behavior
ISuggest Agent shows its read. Looks right · Looks wrong · Edit. No write to Salesforce.
IIApply One-click approve. Earned only after confirm-rate data proves the suggestions are reliable on this field, for this customer.
IIIAuto + Confirm Writes first, surfaces Confirm/Undo — unlocked only after Stage II hesitation data clears. No skipping stages, no per-customer overrides.

Never-auto fields

Stage. Amount. Close Date. Forecast Category. Even after trust is earned. Even when statistically safe. The constraint is structural, not configurable.

Manager loop-in is the accountability layer

High-severity alerts post into the deal Slack thread the manager is already in. The agent decides what's a gap. The team decides the response. The manager being in the thread means the rep can't quietly skirt it.

Every claim has a source. Every source has a timestamp.

Quoted sentence, exact timestamp, named speaker. Paraphrased evidence is a trust failure. Every surface — pre-call brief, manager DM, recap email — renders the same five-element sequence: risk, move, evidence, temporal proof, accountable decision.
